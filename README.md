## Setup

```sh
$ make install
```

## Run linters

```sh
$ make lint
```

## Build project

```sh
$ make build
```

## Run development environment

Init local server with autoreload of browser, compile sass to css, pug to html

```sh
$ make dev
```

## Deploy

```sh
$ make deploy
```
